# Student-Mental-Health-Data-Analysis (Interactive Dashboard creation using MS Excel)

## Project Objective 
This project analyzes student mental health data using Microsoft Excel. The dataset includes variables such as course of study, gender, age, CGPA, and marital status, alongside mental health indicators like depression, anxiety, panic attacks, and help-seeking behavior.
Pivot Tables and charts were used to explore patterns and compare how mental health conditions vary across different student groups. The analysis highlights key trends and provides insights into the relationship between academic, demographic factors, and mental health, as well as the gap between experiencing issues and seeking professional support.

## Questions (KPIs)
-	Which course has the highest number of students with depression?
-	Which gender is more affected by depression?
-	Which age group experiences the highest level of depression?
-	Do students with lower CGPA have higher depression levels?
-	What percentage of depressed students seek professional help?

## Process
- Collected student data including course of study, gender, age, CGPA, marital status, and mental health indicators (depression, anxiety, panic attacks, and help-seeking behavior). 
- Ensured consistency in responses (e.g., Yes/No format), removed duplicates, and corrected inconsistencies in course names. 
- Organized the dataset in Excel and grouped variables such as age and CGPA for better analysis. 
Used Pivot Tables to examine relationships between mental health conditions and factors like course, gender, age, and academic performance. 
- Created charts (clustered and stacked column charts) to clearly present patterns and comparisons. 
- Designed a simple interactive dashboard using charts and slicers for better data exploration. 
- Identified key trends and summarized findings on student mental health and help-seeking behavior.
